### 内部使用邮件脚本工具
包含登录授权及两类邮件的自动抓包获取需要数据构建邮件内容发送，仅内网可使用

可执行文件支持平台:
* Windows
* Mac OSX
* Linux

进入[Github Release](https://github.com/wuseal/DT-Email-Tool/releases) 界面即可下载相应系统支持的版本
